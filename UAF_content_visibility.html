Mecanismo Original (Dark-life944):
1. Define content-visibility: hidden no container pai → altera o layer tree
2. Remove o elemento filho → libera memória associada ao layer
3. Restaura content-visibility: auto → dispara recomputação do layer tree
4. Heap spray com Uint8Array para controlar a memória liberada
5. MutationObserver re-dispara o UAF ao detectar mudanças

Variantes adicionais neste PoC: Múltiplos filhos, CSS animations simultâneas, e will-change spray.

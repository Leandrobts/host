03 — Exception Boundary — Onde o Engine Verifica (e Onde Não Verifica)

🔍 O QUE OBSERVAR:
• PASSOU SEM EXCEÇÃO onde esperava → engine não verifica esse caso = superfície de bug
• valor retornado em vez de exceção → o que está naquele endereço de memória?
• tipo de exceção → TypeError vs RangeError diz qual path de código foi atingido
• crash em vez de exceção → o engine nem chegou a verificar
• undefined em vez de exceção → leitura de memória não inicializada

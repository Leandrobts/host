CSS Nesting Hoisting — Style Resolver UAF
🔧 Safari 18.2 Release Notes · Fix: 300297@main

Safari 18.2 removeu o "hoisting behavior" do CSS Nesting onde seletores pai eram sempre embrulhados em :is(). No PS4 (Safari 17 branch), este comportamento ainda existe e pode ser abusado para crashes em seletores :has() + nesting complexo.

🔧 WebKit commits 2024 — múltiplos fixes silenciosos
● RESOURCE EXHAUSTION — Segmentos Unicode OOM
Intl.Segmenter com textos extremamente longos + grapheme boundaries de scripts raros (Tibetano, CJK extendido, Emoji multi-part) causa OOM/crash no WebKit 17 branch. Múltiplos fixes silenciosos sem CVE no período 2023-2025.
SEM CVE NO-JIT ✓ Técnica nova
WebProcess Chain: RegExp + Proxy + WeakRef
🔬 Pesquisa original — combinação de features Safari 17
● COMBINAÇÃO DE FEATURES — State machine confusion
Combinação de: RegExp com named lookbehind (bugzilla 274053) + Proxy handler com WeakRef tracking + GC forçado. Cria condição de race entre GC e Proxy target collection. Safari 17 branch pode ter bug na interação — não testado na cena PS4.

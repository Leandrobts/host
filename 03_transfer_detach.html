03 — ArrayBuffer.transfer: Detach, Stale Views, OOB

✓ Log v8/T10 confirmou: ArrayBuffer.transfer disponível (ES2024) neste WebKit.
🔍 O QUE OBSERVAR:
• crash ao acessar view de buffer transferido = use-after-free
• valor não-zero em view de buffer detachado = leitura de memória stale
• transferSize maior que original → novo buffer usa nova região de memória
• TypedArray após transfer: byteLength=0 mas [i] retorna undefined (spec correto)
• DataView após detach: deve lançar TypeError — se não lançar, é bug

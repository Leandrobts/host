PS4 // CRASH POC v2
—
Categorias
DOM Stress
Testes de limites do parser DOM e do motor de layout do WebKit. Cada teste foca em uma edge case específica que versões antigas do WebKit não tratam corretamente.
CSS Parser
O CSS parser do WebKit do PS4 tem limites conhecidos com seletores aninhados profundos, calc() em cascata e media queries malformadas.
Memory Pressure
Força o motor JavaScript a alocar e descartar memória de formas que podem travar o GC ou saturar o heap do PS4.
JS Engine
Edge cases no JavaScriptCore do PS4 — recursão, coerção de tipos extrema, e operações que deveriam crashar mas talvez não crashem.
Media e Canvas
O pipeline de mídia e canvas do PS4 tem limitações de hardware — canvas extremo, WebGL edge cases e media malformada.
Combinados
Combinações simultâneas de DOM + CSS + memória que juntas podem travar o compositor do PS4 mesmo quando separadas não causam crash.

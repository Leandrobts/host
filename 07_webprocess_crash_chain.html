🟡 Bug #7 — Chain: RegExp Named Lookbehind + Proxy + WeakRef GC Race

Pesquisa Original · Combinação de features Safari 17 · Sem CVE · Não testado na cena PS4

🔬 TÉCNICA ORIGINAL — Combinação de três features com interação não testada RegExp named lookbehind → cria objetos de captura com live binding
Proxy com handler → intercepta acesso às named capture groups
WeakRef tracking → mantém referência fraca aos objetos de captura

RACE CONDITION:
1. RegExp.exec() retorna match array com named groups
2. Proxy wraps o match array → intercepta acesso a .groups
3. GC é forçado DURANTE o callback do Proxy
4. WeakRef para o match object pode retornar undefined
5. Se Proxy handler acessar .groups depois → objeto coletado → UAF potencial

Safari 17 tem implementação parcial de lookbehind — Bugzilla 274053 relata comportamentos
inesperados em patterns complexos. Nenhum writeup PS4 menciona esta técnica.

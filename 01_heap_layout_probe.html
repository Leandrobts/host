01 — Heap Layout Probe — Tamanho de Objetos, Alinhamento, Gaps

🔍 O QUE OBSERVAR:
• delta(a,b): distância entre dois objetos consecutivos → tamanho + overhead do alocador
• alinhamento: se todos os endereços são múltiplos de 0x10 (16), 0x20 (32), etc.
• consistência: se os deltas são iguais entre execuções → alocador determinístico
• valor hex de qualquer leitura OOB → copiar e reportar
• crash: anotar o último T00 visível antes do crash

RELEVÂNCIA PS4: Branch Safari 17 (Set/2023) → ANTERIOR ao fix de Out/2024
Mecanismo: O CSS Anchor Positioning está em desenvolvimento ativo no WebKit. Quando um popover com anchor está sendo escondido via display:none ou popover.hidePopover(), o StyleTreeResolver tenta processar o elemento para anchor positioning, mas o renderer() retorna nullptr porque o elemento já está sendo destruído. O null deref causa crash no WebProcess. O PS4 usa o branch anterior ao fix — este crash nunca foi testado na cena PS4.
